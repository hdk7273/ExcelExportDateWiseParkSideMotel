This is simple xls generator for Parkside motel.



To generate file:
1. This solution has file in it ReadyChange.xls used for template and generate output file at:
C:\Users\Public\Downloads\ExcelGenerator\01_YYYY
2. After file got generated for print need to upload on Google Drive:
https://drive.google.com/drive/folders/19rqcHtFcHw0wR2N3Crpx74LibQ6bmGfs?usp=drive_link
3. Open in Google Sheet online and generate pdf to print
4. while setting up file we need to set fit to page and Top 0.357" Left/Right 0.432" Bottom 0.507"
5. then generate pdf or print directly
